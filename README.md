# Tic Tac Toe (Python CLI Game)

A simple **command-line Tic Tac Toe game** built with Python, featuring a basic AI opponent.

The project demonstrates core programming concepts such as game loops, condition checking, and simple AI decision-making.

---

## 🎮 Features

- Classic Tic Tac Toe gameplay in terminal
- Player vs Computer mode
- Basic AI logic:
  - Try to win if possible
  - Block player winning moves
  - Otherwise choose strategic positions
- Colored terminal output for better UX
- Input validation for user moves
- Win / lose / draw detection

---

## 🧠 Game Logic Highlights

- Board represented as a 1–9 indexed list
- Winning conditions defined using predefined tuples
- Move validation ensures only empty cells are selected
- AI prioritization strategy:
  1. Winning move
  2. Blocking opponent
  3. Preferred move positions

---

## 🛠 Tech Stack

- Python 3
- termcolor
- colorama

---

## ▶️ How It Works

- Player chooses a number (1–9) to place "X"
- Computer automatically responds with "O"
- Game continues until:
  - Player wins
  - Computer wins
  - Board is full (draw)
